# easysocks
Code for Research/Development Purposes.

实验目标
1. 用asyncio实现一个socks5服务器。在版本734c38b102d56dfe7dd1435fbd84a1156143223e中已实现。
2. 加入forward_server端,在加入forward_server端和proxy_server端之间实现无加密的forward。 在a195c324528ebc378fa8de1323725d8f776c971c版本中已实现。
3. 实现forward_server和proxy_server之间的多路复用，所有来回的数据都在一个socket连接上传输，目的是避免多次建立tcp的资源消耗。
4. 用TLS在forward_server和proxy_server之间传输数据。
5. 添加一个简单的用户身份验证功能，在proxy_server端实现用户身份验证。
